# Unicode NPM Packages

-   [@unicode/unicode-17.0.0](https://www.npmjs.com/package/@unicode/unicode-17.0.0)

    Complete Unicode data package.

-   [regenerate](https://github.com/mathiasbynens/regenerate)

    A Unicode-aware regular expression generator.  Generates
    ES5-compatible regular expressions for sets of Unicode symbols or
    code points.  This is "natively" tricky because in JavaScript,
    strings are UTF-16 and astral plane characters (U+10000 and above)
    are implemented as surrogate pairs.
    
-   [Specimen Skeleton](https://github.com/googlefonts/specimen-skeleton)

    To generate font specimens, you clone this repository and run
    `yarn` commands from in that clone.

-   [specimen-skeleton-support](https://github.com/kabisa/specimen-skeleton-support)

    The tools used by Specimen Skeleton to generate typeface specimens.

-   [Specimen Tools for Websites](https://github.com/graphicore/specimenTools)
